# vue-back4app-docker

Proyecto base de Vue 3 con Vite para la primera etapa del tutorial de Back4App sobre despliegue y dockerizacion de una aplicacion Vue.js.

## Tecnologias iniciales

- Vue 3
- Vite
- JavaScript
- npm

## Instalacion

```bash
npm install
```

## Ejecucion local

```bash
npm run dev
```

Para generar una version de produccion:

```bash
npm run build
```
